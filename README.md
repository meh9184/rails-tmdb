# rails-tmdb
TMDB를 이용한 영화, 드라마, 인물 DB 구축

## Dependency
+ RVM https://rvm.io/
+ Ruby 2.5.3 
+ Bundler https://rubygems.org/gems/bundler/versions/2.0.1
+ Rails 5.2.2 https://rubygems.org/gems/rails/versions/5.2.3
+ MySQL 5.5~5.6
